# 시(물리)에이션

물리학 전반을 다루는 교육용 물리 시뮬레이션 웹 플랫폼입니다.  
역학·에너지, 전자기, 광학, 양자역학, 상대성이론 등의 개념을 인터랙티브 시뮬레이션으로 탐구할 수 있습니다.

---

## 폴더 구조

```
시뮬레이션/
├── index.html              # 랜딩 페이지
├── main.html               # 시뮬레이션 목록 메인 페이지
├── 시뮬레이션.html          # (레거시) 메인 페이지
├── admin.html              # 관리자 페이지
├── main.js                 # 메인 스크립트
├── firebase-auth.js        # Firebase 인증
├── config.js               # 설정
├── styles.css              # 전역 스타일
├── sitemap.xml
├── CNAME
├── space-bg.jpg
│
├── simul/                  # 시뮬레이션 HTML 파일 모음
├── data/                   # 데이터 (quiz-data.js 등)
├── api/                    # 서버리스 API
├── muligo/                 # 물리고 게임 (복도 탐험 RPG)
├── rank/                   # 랭킹 시스템
├── image/                  # 공통 이미지 (마스코트, 파비콘 등)
└── thumb_nail/             # 시뮬레이션 썸네일 이미지
```

## 주요 서브 프로젝트

### 물리고 (`muligo/`)
복도 탐험 RPG 스타일의 물리 학습 게임입니다.  
`index.html` → `app.js` → `missions/`, `scenes/` 구조로 구성됩니다.

### 랭킹 시스템 (`rank/`)
사용자 점수 및 티어(아이언 ~ 챌린저)를 관리합니다.
